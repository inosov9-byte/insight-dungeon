# Insight Dungeon

A dungeon-crawler game where your analytical thinking defeats monsters, not button mashing.

**[Play it live](https://inosov9-byte.github.io/insight-dungeon/)**

## How It Works

1. Choose a class (Analyst, Strategist, Provocateur)
2. Face business traps disguised as dungeon monsters
3. Write your approach to handle each trap
4. An AI scores your response (1-10) — weak insights deal damage to you, sharp ones defeat the trap
5. Survive all encounters to clear the dungeon

## Why This Exists

Most "learning experiences" are passive. Watch a video. Click through slides. This is active — you have to think, and the thinking gets evaluated.

Built as a prototype for exploring AI-scored learning at scale.

## Tech

- Single HTML file, no dependencies
- Claude API (Haiku) for real-time scoring
- Web Audio API for sound effects
- Runs entirely in browser
